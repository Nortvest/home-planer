# Frontend

Статический HTML/CSS/JS. Без бандлеров и фреймворков.

## Структура

```
frontend/
├── index.html          — календарь (основной экран)
├── admin.html          — админка (пользователи и шаблоны)
├── dashboard.html      — дашборд со статистикой
├── css/
│   ├── base.css        — сброс стилей, CSS-переменные
│   ├── theme.css       — палитры светлой/тёмной темы
│   ├── calendar.css    — сетка календаря
│   ├── card.css        — карточка задачи
│   └── admin.css       — админ-панель
└── js/
    ├── config.js       — адрес бэка (ЕДИНСТВЕННОЕ место)
    ├── api.js          — обёртка над fetch
    ├── state.js        — общее состояние
    ├── calendar.js     — рендер сетки
    ├── card.js         — карточка + дропдаун
    ├── admin.js        — админ-логика
    ├── dashboard.js    — дашборд
    └── theme.js        — переключатель темы
```

## Запуск (dev)

Любой статический сервер:

```bash
python -m http.server 5500
```

Затем откройте `http://localhost:5500`.

## Настройка адреса бэка

Отредактируйте `js/config.js`:

```js
const API_BASE = 'http://localhost:8000/api';
```

## Деплой

В продакшене бэкенд отдаёт статику фронта. Установите `SERVE_FRONTEND=1` на бэке — фронт будет доступен по основному адресу.
